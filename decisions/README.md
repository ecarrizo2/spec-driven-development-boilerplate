# Architecture Decision Records

Lightweight decision log for this hub. Each ADR captures a conclusion — not a debate.

## Format

```
# ADR-NNN: Title

**Status:** Accepted | Proposed | Superseded
**Date:** YYYY-MM-DD

## Conclusion

What we decided, in 1-3 sentences.

## Why

Bullet list of reasons.

## Consequences

What this means in practice.
```
